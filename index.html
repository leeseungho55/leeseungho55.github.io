희망직무

Back-end 개발자(DB, API, JAVA기반)로서 기술적 도전과 성장을 이루고 싶습니다. 백엔드 개발은 데이터베이스 관리, 서버 로직 구현, API 설계 등 다양한 기술적 도전을 제공하여, 문제 해결 능력을 향상시키고 성취감을 느낄 수 있습니다.

시스템 안정성과 성능 최적화에 기여하고 싶습니다. 백엔드 개발은 애플리케이션의 안정성과 성능을 책임지며, 효율적인 코드 작성과 시스템 최적화를 통해 사용자에게 안정적이고 빠른 서비스를 제공하는 것을 목표로 합니다.

데이터 처리와 분석에 흥미가 있습니다. 데이터베이스 설계와 데이터 처리, 분석을 통해 데이터의 가치를 극대화하고, 이를 통해 비즈니스 인사이트를 제공하며 백엔드 개발의 매력을 느끼고 있습니다.

Project Description

프론트엔드 웹 프로젝트

프론트엔드 웹 프로젝트인 범죄 회피 장애인 네비게이션 웹 버전은 장애인들이 도로를 안전하게 이용할 수 있도록 설계된 애플리케이션입니다. 사용자는 실시간으로 업데이트되는 범죄 데이터를 기반으로 안전한 경로를 선택할 수 있습니다. 네비게이션 기능은 일부 문제가 발생하고 있지만, 마커 추가 및 API 연결 기능은 원활히 작동하여 사용자에게 정확한 정보를 제공합니다. 이 프로젝트는 사회적 약자의 보호와 안전을 증진하기 위한 목적으로 개발되었습니다.

백엔드 웹 프로젝트(해커톤대회-멋쟁이사자처럼)

멋쟁이사자처럼 프로그램에서 진행한 노인 건강 웹 프로젝트는 백엔드 개발에 중점을 두었습니다. 사용자가 건강 관리를 위해 필요한 정보와 리소스를 제공하며, 주요 기능으로는 사용자 맞춤형 건강 정보 추천 시스템과 건강 상태 모니터링 기능을 포함합니다. 백엔드는 데이터베이스 관리, API 연동 및 데이터 분석을 통해 사용자에게 정확하고 신속한 서비스를 제공하며, 노인들의 건강 증진에 기여하는데 목적을 두고 있습니다.

백엔드 웹 프로젝트(해커톤대회-멋쟁이사자처럼2)

멋쟁이사자처럼 운영진들이 세종의 해커톤 대회에서 개발한 농촌봉사 플랫폼 백엔드는 농촌 지역과 봉사자를 연결하는 주요 기능을 가지고 있습니다. 이 플랫폼은 농업 환경에서 필요한 봉사자와 자원봉사를 지원하며, 백엔드 개발은 데이터 관리와 봉사자-농촌 사이의 원활한 소통을 위한 API 설계에 중점을 두었습니다. 사용자들은 편리하게 봉사 기회를 찾고 등록할 수 있으며, 농촌 사회 발전에 기여하는 플랫폼을 제공합니다.

모바일 프로젝트

이 안드로이드 앱은 장애인을 위한 범죄 회피 네비게이션으로, 실시간 범죄 데이터를 기반으로 사용자의 안전을 최우선으로 합니다. 앱은 사용자가 안전한 경로를 선택할 수 있도록 실시간으로 위험 지역을 식별하고 경고합니다. 사용자가 위험 지역에 접근할 때는 진동과 경고 알림을 통해 경각심을 높이며, 긴급 상황 발생 시 위치 기반으로 긴급 전화를 즉시 호출할 수 있습니다. 이를 통해 장애인의 안전한 이동을 돕고 사회적 포용성을 증대시킵니다.

향후 예정 프로젝트(지능형 도시 교통 관리 시스템)

향후 예정 프로젝트로는 지능형 도시 교통 관리 시스템을 개발하는 것입니다. 이 시스템은 실시간 교통 데이터 수집과 분석을 통해 도로 혼잡도를 예측하고, 최적 경로를 제안하여 교통 흐름을 최적화합니다. 또한, 차량간 통신 기술을 활용하여 사고 예방 및 신속한 대응을 가능하게 하며, 환경에 미치는 영향을 최소화하는 친환경 교통 인프라를 구축하는 것을 목표로 합니다..
